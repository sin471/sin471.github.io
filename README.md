# sin471.github.io

個人ポートフォリオサイト

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Velite (Markdownコンテンツ管理)

## Development

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で開発サーバーが起動します。

## Build

```bash
npm run build
```

静的サイトが `out/` ディレクトリに生成されます。

## Deployment

mainブランチへのpushで自動的にGitHub Pagesにデプロイされます。
